# ForkTales - Frontend

Frontend application for **ForkTales**.

## Features
- Responsive Design
- Clean UI
- Easy Navigation
- PWA

## Tech Stack
- React
- Vite
- Tailwind CSS

## How to Run
```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```
