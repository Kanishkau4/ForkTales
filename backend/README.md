## ForkTales - Backend

Backend application for **ForkTales**

## Features
- Fast API
- JWT Authentication
- Supabase Database

## Tech Stack
- Python 3.10
- FastAPI
- Supabase (PostgreSQL)

## How to Run
```bash
# Install dependencies
pip install

# Run the development server
uvicorn main:app --reload
```
